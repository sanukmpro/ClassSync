Relevant Truths

The logic and architecture behind the synergy.

AuthGate Protocol

ClassSync implements a robust AuthGate logic. It seamlessly checks Firebase authStateChanges to route users either to Role Selection or their persistent Dashboard based on cached credentials.

Dynamic Firestore Streams

Your library is powered by high-speed Firestore Query Snapshots. It filters materials by Department and Semester in real-time, ensuring zero-latency resource searching.

Cloud Messaging (FCM)

ClassSync utilizes Firebase Messaging for foreground and background notifications. Faculty announcements and Mailbox alerts reach students instantly across Android and Windows.

Teacher ID Request

Teachers aren't just registered; they're verified. Our system creates a secure request document in Firestore, requiring admin approval before faculty tools are unlocked.

Multi-Platform Persistence

Designed for Android, Web, and Windows. We use Platform-aware logic and SharedPreferences to maintain user sessions and theme settings (Dark/Light mode) permanently.

In-App Preview Engine

Supports PDF, Images, and Documents. ClassSync intelligently decides between an in-app image viewer or an external application launch for seamless file reading.
